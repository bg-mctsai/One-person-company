# 前情提要腳本（死前跑馬燈版本 - 懸疑推理版）

## 呈現方式說明

**呈現方式**：靜態圖片片段 + 內心獨白 + 音效（無動畫、無影像）
- **跑馬燈效果**：每個片段出現後，慢慢縮小消失，再出現下一個片段
- **片段設計**：每個片段包含關鍵線索（關鍵字、手上拿的東西、說的話、背景的字）
- **旁白**：第一人稱內心獨白，自己對自己說的話
- **音效**：環境音效、懸疑音效、背景音樂
- **系統語音**：在所有場景結束、一切變黑暗之後才出現

**時間軸**：不強制2分鐘，根據內容自然流動（約1.5-2.5分鐘）

**線索設計原則**：
- 每個片段必須包含可觀察的線索（關鍵字、物品、對話、背景文字）
- 線索必須與後續時刻連貫，認真觀看的你能獲得優勢
- **聚焦辦公室霸凌**：所有場景都在辦公室，線索與辦公室霸凌直接相關
- **自救主題**：前情提要建立「被榨乾」的憤怒與清醒，為「重來一次把界線拿回來」鋪墊

**重要設定**：
- 你在 2024 年 **過勞猝死**（不是意外、也不是單一壞人「害死你」）
- 其他角色不記得5年後發生的事（他們沒有回到過去）
- 只有你知道未來的悲劇
- 你可以改變自己的行為，但不能用「他們記得未來」來逼供（因為其他人不記得）
- 目標：改變自己的行為模式（核心）+ 找出「壓垮你的機制」（副線）
- 結局選擇：看懂結構後，可以選擇揭露、離開或和解（含自我和解）
- 真相不是抓兇手：你要看懂「**責任傾倒 / 群體風向 / 制度話術**」如何把你推到死路，並看見你自己的「過度討好/不敢拒絕」怎麼被利用
- 角色齒輪補齊：**林予衡（制度守門人）**、**何婉柔（輿論放大器）** 會以「影子/道具/群聊」方式出現在前情提要，讓玩家在進主線前就能先看到結構的輪廓

**角色命名（正式版）**：
- 主角：予安
- 主管：王明哲
- 同事（試探/搶功）：許子維
- 同事（輿論放大器）：何婉柔
- 同事（辦公室戀情）：周承翰
- 人資：林予衡
- 系統：裁決系統
