# 完整故事情節設計（職場自救推理版）

## 故事時間軸

**設定**：玩家在 2024 年 **過勞猝死**，回到 2019 年（5 年前）
**目標**：透過**精簡10個關鍵時刻（主線）**，改變自己的行為模式，**看懂壓垮你的機制**，改變命運  
> 推理密度保留：精簡版會在少數時刻一次產出 2 條線索（對齊 `挑戰系統.md`）。

**故事核心**：從認識到被榨乾走向深淵，但**你不知道是哪個節點開始失控**
- **聚焦**：辦公室的人情世故（同事A、主管）與私領域被侵入（同事C）
- **所有場景都在辦公室**，線索與辦公室霸凌直接相關
- 2019年：認識這些人，他們一開始對你很好
- 2019年7月開始：責任開始傾倒、風向開始形成，你一次次把界線讓出去
- 2020年：被持續霸凌，走向深淵，線索逐漸浮現
- 2024年：過勞猝死（不是意外，也不是單一壞人）

**重要設定**：
- 其他角色不記得5年後發生的事（他們沒有回到過去）
- 只有玩家知道未來的悲劇
- 玩家可以改變自己的行為，通過改變行為模式來「翻盤/自救」（因為其他人不記得，但你可以改變過去）

**前情提要關聯**：
- 前情提要中的每個片段都包含關鍵線索（關鍵字、手上拿的東西、說的話、背景的字）
- 認真觀看前情提要的玩家能在後續挑戰中獲得優勢
- 所有線索都與後續挑戰連貫，形成完整的推理鏈

**懸疑核心**：
- **你以為是「某個人」在針對你，但你抓不到單一壞人**
- **每個挑戰都有線索，指向「責任傾倒/風向/制度話術」如何把你推進死路**
- **你需要收集線索，分析：到底是哪個節點開始失控？你每次是怎麼把界線讓出去的？**

## 遊戲目標設計（混合導向）

### 三層目標設計

#### 第一層：表面目標（玩家以為的目標）
- 看懂到底是誰在把工作丟給你、誰在帶風向
- 讓「不合理」回到事實與邊界（翻盤）
- 改變自己的命運

#### 第二層：實際目標（遊戲機制）
- **改變自己的行為模式**（核心目標）
  - 不再過度討好
  - 學會設定界線
  - 為自己發聲
  - 建立內在力量
- **看懂壓垮你的機制**（副線目標）
  - 收集線索
  - 分析結構
  - 找到關鍵節點（誰丟鍋/誰放大/誰背書/哪個流程讓你永遠說不清）
- **看懂結構後，選擇如何處理**（結局選擇）
  - 揭露：公開事實與流程漏洞（正面硬幹）
  - 離開：帶著界線離開（不再把命交出去）
  - 和解：不是替他們開脫，是和解自己（停止自我傷害循環）

#### 第三層：深層目標（故事主題）
- **自我救贖**：從依賴他人到接住自己
- **成長**：從被動受害到主動反擊
- **真相**：從未知到真相，從被背叛到獲得正義

### 遊戲機制設計

#### 雙層挑戰系統
每個挑戰都有兩個層面：

**1. 行為層面（核心）**
- 改變自己的行為模式
- 不再過度討好、學會設定界線、為自己發聲
- 影響：改變與他人的關係，自然改變命運

**2. 推理層面（副線）**
- 收集線索，找出真相
- 觀察每個人的行為，尋找可疑的線索
- 影響：看懂「壓垮你的機制」，為結局選擇做準備

**兩個層面的互相影響**：
- 改變行為後，更容易收集線索（因為你更自信、更警覺）
- 收集線索後，更容易改變行為（因為你看懂機制怎麼運作）
- 兩個層面互相促進，形成完整的遊戲體驗

#### 行為改變機制
- **過度討好 → 設定界線**：學會說「不」，保護自己的權益
- **害怕衝突 → 為自己發聲**：學會表達自己的需求，爭取應得的認可
- **依賴他人 → 建立內在力量**：學會接住自己，不再依賴他人
- **被動受害 → 主動反擊**：學會主動保護自己，揭露真相

#### 線索收集機制
- **前情提要中的線索**：每個片段都包含關鍵線索
- **挑戰中的線索**：每個挑戰完成後，玩家會獲得線索
- **線索類型**：隱藏線索、關鍵線索、累積結果、最終線索
- **線索分析**：玩家可以查看收集的線索，分析線索之間的關聯

## 角色背景與動機（關係與壓力檔案）

### 同事A（工作傾倒者？）
**背景**：比你早入職2年，原本是部門的明日之星，但你的到來威脅到他的地位。
**表面動機**：害怕你取代他，所以先示好建立信任，再逐步霸凌你。
**性格**：表面友善，內心嫉妒，善於操縱人心。
**弱點**：害怕被超越，缺乏安全感。
**可疑線索**：
- 過度熱情，主動接近你
- 記住你的工作方式和弱點
- 在關鍵時刻總是「剛好」出現
- **但**：他可能只是被利用的工具
**環境的暗中導引**（讓他覺得「丟給你很合理」）：
- 用微暗示放大他的不安：「你覺得他是不是太想表現？」
- 用績效/升遷壓力讓他把你當成工具與替罪羊

### 同事C（私領域被利用？）
**背景**：辦公室同事，以「關心/懂你」的姿態靠近，讓你在壓力裡把情緒與時間交出去。
**表面動機**：不一定惡意，但習慣把需求丟給「不會拒絕的人」；你越承接，他越覺得理所當然。
**性格**：表面溫柔，核心是逃避責任、怕麻煩。
**弱點**：怕衝突、怕被要求承擔。
**可疑線索**：
- 記住你的情感弱點
- 在關鍵時刻總是「剛好」出現
- 與其他嫌疑人可能有聯繫（辦公室霸凌計劃）
- **但**：他可能只是被利用的工具

### 主管（制度背書者）
**背景**：部門主管，重視效率和結果，容易被表面現象迷惑。
**表面動機**：被同事A誤導，認為你能力不足，所以配合同事A的行為。
**性格**：表面公正，但容易被影響，缺乏判斷力。
**弱點**：過度信任下屬，不願意深入調查。
**可疑線索**：
- 在關鍵時刻總是「剛好」出現
- 可能知道更多內幕
- 可能與其他嫌疑人串通
- **但**：他可能只是被利用的工具
**流程話術**（把不信任包成風險控管）：
- 選擇性提供「對你不利的片段」（情緒爆點、被截斷的對話、會議紀要）
- 反覆灌輸標籤：情緒化 / 不服管理 / 影響團隊 / 風險

### 同事B（同事群代表 / 輿論放大器）
**背景**：資深同事（或行政/PM 型角色），熟悉公司八卦與內部流程，掌握群聊與茶水間話語權。
**表面動機**：怕惹麻煩、想自保，所以「跟著多數說法走」，把轉傳當成中立。
**性格**：社交手腕強、嘴甜、擅長包裝訊息成「大家都這樣覺得」。
**弱點**：怕自己成為下一個被切割的人；怕被貼上「不合群/不挺公司」標籤。
**可疑線索**：
- 你每次被貼標籤後，群聊裡總有「剛好」的截圖/片段被擴散
- 對你的評語永遠有「聽說」「大家說」「我也是為你好」的語氣
- **但**：他可能只是被風向推著走，也可能被 HR 明示/暗示要站隊
**制度的暗中導引**（把個人惡意變成群體共識）：
- HR不下命令，只丟一句：「最近有人反映…先依流程處理。」
- 同事B把這句話翻譯成群聊的「社會證明」：大家都覺得你有問題

### HR（制度守門人 / 程序至上）
**身份**：你現在公司的HR
**定位**：不是「真兇」，也不需要「害你」——他只要把你當成流程上的一個 case。
**行為特徵**：
- 永遠先問「有沒有紀錄/證據/流程」，而不是先問「事實是什麼」
- 用模板語言把人的痛變成「可受理範圍」
- 在關鍵節點出現（考核/申訴/改善計畫/調整職務），但不一定帶惡意
**玩家可拆點**：把情緒戰改成「事實＋責任邊界＋程序」戰（存證、交付範圍、會議紀錄、時間線）

## 霸凌鏈條（動機與手段，讓行為「合理」）

> 核心原則：每個人都要有「自己的理由」（自保/省事/求績效/怕麻煩）＋「制度環境提供的遮羞布」（流程/KPI/共識/慣例），玩家才有拆解空間。

- **HR（制度守門）**：不是陰謀家，但流程永遠先於人；關鍵節點只看紀錄，讓你所有的痛都變成「不在受理範圍」。
- **同事A（傾倒）**：嫉妒＋搶功立功（自己的壞念頭）＋省事 → 把最髒最急的工作丟給你，測你會不會吞。
- **主管（背書）**：KPI/怕麻煩（自己的壞念頭）→ 默許不合理分工，用「程序」壓你，把失敗變成你的責任。
- **同事C（私領域侵入）**：以關心索取、逃避承擔 → 把情緒/雜事丟給你，讓你更難拒絕工作傾倒。
- **同事B（放大 / 同事群代表）**：從眾/自保/八卦 → 把片段包成「大家都這樣覺得」，讓你失去立足點。（同事群背景存在，但由同事B具體出手）

## 線索系統設計

### 線索類型
1. **直接線索**：明確指向某個嫌疑人的證據
2. **間接線索**：需要推理才能理解的線索
3. **誤導線索**：指向錯誤方向的線索
4. **關鍵線索**：揭露真相的關鍵證據

### 線索收集機制
- **前情提要中的線索**：每個片段都包含關鍵線索（關鍵字、手上拿的東西、說的話、背景的字）

- 線索可能隱藏在對話、行為、場景中
- 玩家需要仔細觀察和分析
- 收集足夠線索後，可以推斷出「壓垮你的機制」

- **場景一-片段4** → 挑戰6：主管的「閃爍眼神」可能是知情
- **場景一-片段5** → 挑戰7：同事A的「害怕眼神」可能是被威脅
- **場景一-片段6** → 挑戰8：同事A的「點頭示意」可能是串通

- **場景一-片段8** → 挑戰12：同事A的「害怕」可能是被威脅（關鍵線索）
- **場景二-片段1** → 挑戰4：同事C的「發送訊息」可能是通報
- **場景二-片段2** → 挑戰5：同事C的「閃爍眼神」可能是說謊
- **場景二-片段3** → 挑戰9：同事C的「閃爍眼神」可能是說謊
- **場景二-片段4** → 挑戰10：同事C的「發送訊息」可能是通報

### 職場類（3個挑戰）

#### 挑戰1：初入職場的歡迎（2019年1月10日）

**故事背景**：
- 這是回到5年後的第一個挑戰
- 你剛入職，同事們對你很熱情
- 這是美好的開始，但也是陷阱的開始
- **前因**：你剛畢業，對職場充滿期待，渴望被接納
- **後果**：如果你完全信任，之後會被利用；如果保持警覺，可以提前防範

**具體情境**：
- 你第一天上班，主管和同事A對你很熱情
- 主管說：「歡迎加入團隊！我們很期待你的表現。」
- 同事A主動走過來，伸出手：「我是同事A，有什麼問題隨時問我，不用客氣。」
- 他帶你熟悉環境，介紹工作內容，甚至分享了一些「內部消息」
- 你感覺很溫暖，覺得找到了好同事
- **但你注意到**：同事A的眼神有些奇怪，似乎在觀察你的反應
- **隱藏線索**：同事A的「過度熱情」可能是偽裝

**情感細節**：
- 你感到被接納的溫暖
- 你對未來充滿希望
- 你開始信任這個環境
- **但內心深處**：有一絲不安，因為你不知道「不合理會從哪裡長出來」

**挑戰目標**：
- 建立良好的職場關係
- 但要保持警覺，不會完全信任
- 觀察每個人的行為，尋找可疑的線索

**改變後的結果**：
- 你建立良好的職場關係
- 但你會保持警覺，不會完全信任
- 你開始注意到一些「異常」的行為
- 你獲得**線索1**：同事A的「過度熱情」

**線索分析**：
- **線索1**：同事A的「過度熱情」可能是偽裝
- **可能指向**：同事A可能在試探你的底線（你越好說話，越容易被丟工作）
- **需要更多線索**：才能確定

#### 挑戰2：第一次合作（2019年2月15日）

**故事背景**：
- 你和同事A第一次合作專案
- 他對你很友好，分享工作經驗
- 你開始信任他
- **前因**：第一次合作成功，你覺得找到了好搭檔
- **後果**：如果你完全信任，他會記住你的工作方式，之後用來霸凌你

**具體情境**：
- 你和同事A一起完成一個重要專案
- 他對你很友好，主動分享工作經驗：「這個專案我來帶你，你跟著學就好。」
- 他甚至在會議上稱讚你：「這個想法不錯，你很有潛力。」
- 你說：「謝謝你，我學到很多。」
- 他說：「我們是同事，互相幫助是應該的。以後有什麼問題，隨時找我。」
- **但你注意到**：他在稱讚你的時候，眼神有些閃爍，似乎在隱藏什麼
- **隱藏線索**：同事A的「稱讚」可能是試探

**情感細節**：
- 你感到被認可的滿足
- 你對同事A產生信任和依賴
- 你開始把他當成「自己人」
- **但內心深處**：你「知道」這種信任是危險的，但你不知道危險來自哪裡

**挑戰目標**：
- 建立信任關係
- 但要記住這是陷阱，保持警覺
- 觀察每個人的行為，尋找可疑的線索

**改變後的結果**：
- 你建立信任關係
- 但你會保持警覺，不會完全信任
- 你開始獨立思考，不依賴他的「指導」
- 你獲得**線索2**：同事A的「稱讚」可能是試探

**線索分析**：
- **線索2**：同事A的「稱讚」可能是試探
- **可能指向**：同事A可能在收集你的信息
- **與線索1關聯**：兩條線索都指向同事A，但他可能只是被利用

#### 挑戰3：職場聚會（2019年3月20日）

**故事背景**：

## 推理系統設計

### 線索收集機制
- 每個挑戰完成後，玩家會獲得線索
- 線索可能隱藏在對話、行為、場景中
- 玩家需要仔細觀察和分析
- 收集足夠線索後，可以看懂「壓垮你的機制」

### 線索分析系統
- 玩家可以查看收集的線索
- 玩家可以分析線索之間的關聯
- 玩家可以推斷出「壓垮你的機制」與關鍵節點
- 玩家可以選擇把問題怪在某個人身上，或看懂結構

### 真相揭露機制
完成所有**10個挑戰（精簡版主線）**後，玩家需要：

**第一步：行為改變評估**
- 評估玩家是否成功改變自己的行為模式
- 是否不再過度討好、學會設定界線、為自己發聲、建立內在力量
- 影響：決定玩家是否有能力改變命運

**第二步：機制推理**
- 分析所有收集的線索
- 看懂責任傾倒/風向/制度話術的運作方式
- 玩家可以選擇：
  1. **怪某個人（單點歸因）**：看起來很合理，但容易誤判系統性問題
  2. **怪所有人（全面歸因）**：情緒被安放，但缺少可拆解節點
  3. **看懂機制（結構歸因）**：找到「哪個流程/哪句話術」讓你永遠說不清、永遠背鍋

**第三步：結局選擇**
看懂機制後，玩家可以選擇如何處理：
- **揭露路線**：公開事實與流程漏洞，形成可追責的證據鏈（可能付出代價）
- **離開路線**：承認傷害但拒絕再被操控，帶著界線離開（獲得自由）
- **和解路線**：不是替他們開脫，是和解自己（停止自我傷害循環，獲得平靜）
