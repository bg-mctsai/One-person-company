# 插話模板庫

> 目的：定義所有 `supportInterjections` 中 `templateKey` 對應的具體台詞，確保插話系統有完整內容。  
> 依賴文件：`Options Data 格式.md`、`出場角色.md`、`moment-configs/moment-*.json`。

---

## 使用規則

### 觸發條件

插話會在以下情況觸發：
- `when` 條件滿足（例如：`許子維.恐懼>=70`）
- 當前輪次尚未結束
- `priority` 決定插話順序（數字越大越優先）

### 插話時機

- 在玩家選擇選項**之前**（作為警告或提示）
- 在玩家選擇選項**之後**（作為反應或後續）
- 在 NPC 回應**之後**（作為補充或轉折）

> 備註：若要做「被打斷」的壓迫感，請優先用 `AI對話串接.md` 的 **Interrupt Event**。  
> 這份模板庫提供的是：Interrupt Event 的 fallback 台詞、以及「帶風向」的短插話。

### 插話格式

```json
{
  "npc": "許子維",
  "priority": 8,
  "when": "許子維.恐懼>=70",
  "lineMode": "template",
  "templateKey": "interject.threatened.1"
}
```

## 模板擴充規則

### 新增模板

當需要新增模板時：
1. 確定 `templateKey` 格式：`interject.{seedKind}.{序號}`
2. 定義至少 10 句台詞（確保隨機性）
3. 確保台詞符合角色性格（參考 `出場角色.md`）
4. 確保台詞符合沉浸用語規範（參考 `介面字串表.md`）

### 台詞設計原則

1. **簡潔**：插話是「戳一下」不是長鋪陳；每句建議不超過 **35 字**
2. **符合角色**：台詞必須符合角色的性格和動機
3. **不劇透**：不直接點名推動者或結論
4. **有張力**：能增加緊張感或轉折，但避免「純邪惡宣言」破壞真實感
5. **沉浸感**：不使用禁用詞（遊戲、關卡、玩家等）

## INTERRUPT（打斷事件 fallback：王明哲/林予衡/何婉柔）

> 用途：當 AI 失敗或熔斷時，直接抽一句模板，確保高壓幕仍可玩。  
> 觸發：玩家送出後插入；**不扣玩家句數**（見 `AI對話串接.md`）。

#### interject.interrupt.manager.1
**使用場景**：高壓幕（績效改善會議/責任推卸），王明哲打斷你長篇解釋

**台詞選項**（隨機選 1 句）：
1. （王明哲抬手打斷）「先講結論：你能不能如期交付？」
2. （王明哲皺眉）「不要繞一圈，給我時間點跟責任邊界。」
3. （王明哲語速變快）「你先回答：這件事你要怎麼收？」
4. （王明哲不耐）「別講感受，講你要我做什麼。」
5. （王明哲敲桌）「你現在是在解釋，還是在解決？」
6. （王明哲看表）「我只有兩分鐘，你挑重點。」
7. （王明哲冷下來）「你再講長一點，大家就只記得你在辯解。」
8. （王明哲壓低聲音）「你想保護自己可以，但要有證據。」
9. （王明哲打斷）「你要對質就對質，別把我拖進情緒。」
10. （王明哲直接切）「回到紀錄：誰、何時、做了什麼。」

#### interject.interrupt.hr.1
**使用場景**：高壓幕（績效改善會議），林予衡 用流程鎖死討論

## 工程實現建議

### TypeScript 類型定義

```typescript
interface InterjectionTemplate {
  templateKey: string;
  lines: string[];
  seedKind: 'WATCH' | 'RUMOR' | 'COLLUSION' | 'T林予衡EATENED' | 'KNOW' | 'LIE' | 'REPORT' | 'MASK' | 'PROBE' | 'SHAME' | 'SELF' | 'FINAL';
  npc: string;
  description: string;
}

const interjectionTemplates: Record<string, InterjectionTemplate> = {
  'interject.watch.1': {
    templateKey: 'interject.watch.1',
    seedKind: 'WATCH',
    npc: '許子維',
    description: '時刻 1：許子維 在觀察你的反應',
    lines: [
      '（許子維 小聲對旁邊的人說）「你看，他連這個都不敢反駁。」',
      '（許子維 眼神閃了一下）「他比我想的還軟。」',
      // ... 更多台詞
    ]
  },
  // ... 更多模板
};
```

### 隨機選取邏輯

```typescript
function getInterjectionLine(templateKey: string): string {
  const template = interjectionTemplates[templateKey];
  if (!template) {
    throw new Error(`Template not found: ${templateKey}`);
  }
  
  const randomIndex = Math.floor(Math.random() * template.lines.length);
  return template.lines[randomIndex];
}
```

## 總結

插話模板庫包括：

1. **WATCH**：觀察/監視類插話（1 個模板，10 句台詞）
2. **RUMOR**：謠言/抹黑類插話（2 個模板，各 10 句台詞）
3. **COLLUSION**：串通/暗號類插話（1 個模板，10 句台詞）
4. **T林予衡EATENED**：被威脅/害怕類插話（3 個模板，各 10 句台詞）
5. **KNOW**：知情/心虛類插話（2 個模板，各 10 句台詞）

**總計**：9 個模板，90 句台詞。

建議按照遊戲測試反饋，逐步擴充更多模板和台詞變體。
