# 罪語者 (Verbal Sinner) - 核心概念

## 故事主題

**因為人生過得很慘，於是回到5年前，重新開始，並獲得「毒舌系統」的幫助。**

**故事核心**：5年前，你認識了職場同事與情人。他們一開始對你很好，但後來一步步陷害你，讓你走向深淵。**在死亡的最後一刻，你才突然明白：有人害死你，但不知道是誰。**現在你要回到過去，**找出真正的兇手，改變自己的命運。**

## 核心機制

### 1. 前情提要：死前跑馬燈（約1.5-2.5分鐘）

- **呈現方式**：靜態圖片 + 第一人稱內心獨白 + 音效
- **內容**：展現你悲慘人生的關鍵片段（死前跑馬燈效果）
  - 場景一：職場慢性凌遲
  - 場景二：朋友的背叛
  - 場景三：愛情只是交易
  - 場景四：被榨乾的一天（2024年，死亡當天）
  - 場景五：死亡發生（**在最後一刻才意識到：有人害死我，但不知道是誰**）
    - 死因：被車撞死（可選：吃了什麼東西才死、被車撞死、有人推下樓等）
    - 線索：車燈光異常角度、模糊人影、撞擊前的異常等
  - 場景六：你不在之後（**開始有找尋兇手的想法**）
  - 一切變黑暗
  - 系統出現
- **線索埋設**：
  - 在前情提要中埋設線索，讓玩家在死亡瞬間才意識到被陷害
  - 線索包括：車燈光異常角度、模糊人影、撞擊前的異常等
  - 這些線索在死前跑馬燈中快速閃過，暗示與死亡有關
- **情感轉變**：
  - 前面：困惑、憤怒、絕望（以為是自己的錯，以為只是運氣不好）
  - 死亡發生：突然明白（**有人害死我，但不知道是誰**）
  - 之後：決心找出真相（**我要找出真正的兇手**）
- **目的**：建立情感連結，讓你理解為什麼需要重來，並產生「找出兇手」的動機

## 沉浸用語規範（畫面不出戲）

> 規則：整段體驗中，畫面文字 **不使用**「遊戲 / 關卡 / 玩家」等出戲詞。  
> 內部文件可以使用技術詞，但 **所有呈現給你看的文字** 必須遵守以下替換。

- **禁用詞**：遊戲 → **替換**：故事 / 旅程 / 經歷
- **禁用詞**：關卡 → **替換**：時刻 / 片段 / 那一天 / 這一幕
- **禁用詞**：玩家 → **替換**：你 / 我（第一人稱敘事）
- **禁用詞**：挑戰 → **替換**：改寫 / 反駁 / 對話 / 轉折
- **禁用詞**：回合 → **替換**：輪次 / 句數

### 2. 復活機制

- **觸發條件**：你決定復活
- **營收機制**：先看廣告（必須完成）
- **時空設定**：回到5年前（2019年）
- **目標**：
  - 針對被陷害的事件一一反駁，改變職場同事、朋友、情人對你的看法
  - **收集線索，找出真正的兇手**
  - **揭露真相，獲得正義**
- **關鍵時刻數**：14（職場8、情人5、自我1）
- **再試一次**：片段失手 → 毒舌系統羞辱 → 看一次廣告回到 5 分鐘前再試一次（見 `復活機制.md`）
- **全程伴行**：毒舌系統不只在失手出現；復活、每幕進場、你要按下選項前、改寫成立、拿到線索、結算，它都會說話（見 `介面字串表.md` 與 `出場角色.md`）
- **結局評語**（可選）：結算由毒舌系統「宣判」（行為/真相雙軸 + 結局分支評價）
- **成功條件**：走完 14 個關鍵時刻，並找出真正的兇手

### 3. 改寫系統（對話 + 線索）

- **互動方式**：與角色對話
- **線索收集**：每個關鍵時刻完成後，你會獲得線索
- **線索類型**：
  - 直接線索：明確指向某個嫌疑人的證據
  - 間接線索：需要推理才能理解的線索
  - 誤導線索：指向錯誤方向的線索
  - 關鍵線索：揭露真相的關鍵證據
- **推理系統**：
  - 你可以查看收集的線索
  - 你可以分析線索之間的關聯
  - 你可以推斷出真正的兇手
- **目的**：
  - 透過對話改變過去的事件走向
  - **收集線索，找出真正的兇手**
  - **揭露真相，改變自己的命運**

### 4. 懸疑推理系統

- **核心懸疑**：
  - **你不知道誰是真正的兇手**
  - **可能是其中一個，也可能是所有人都有參與**
  - **每個挑戰都有線索，指向不同的嫌疑人**
  - **你需要收集線索，分析誰是真正的兇手**
- **嫌疑人**：
  - 同事A（職場陷害者？/被威脅者？）
  - 情人C（傷害者？/被威脅者？）
  - 主管（幕後黑手？/被威脅者？）
  - 神秘人X（真正的兇手？）
- **真相揭露**：
  - 完成所有關鍵時刻後，你可以選擇找出真正的兇手
  - 所有線索都指向一個「幕後黑手」
  - 真正的兇手可能是一個「幕後黑手」，操縱所有人

## 故事時間軸

- **2019年1月-6月**：認識這些人，他們一開始對你很好（第一階段：建立關係）
- **2019年7月-12月**：他們開始陷害你，但你不知道是誰在背後操縱（第二階段：開始被陷害）
- **2020年1月-6月**：被持續陷害，走向深淵，線索逐漸浮現（第三階段：深淵邊緣）
- **2024年**：死亡，但死因不明
- **死亡的最後一刻**：突然明白「有人害死我，但不知道是誰」
- **回到2019年**：找出真正的兇手，改變自己的命運
