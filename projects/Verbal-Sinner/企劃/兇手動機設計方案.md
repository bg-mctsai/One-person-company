# 兇手動機設計方案（決定版）

> 這份文件是「唯一依據」。**不要再用 ABCD/多選項寫法**，避免劇情與線索系統分歧。

## 0. 一句話總結（給全團對齊用）

**真正的兇手是你現在公司的 HR（神秘人X）**：因為你在 **2016 年**「拒絕替他背書/作證」並丟下一句標籤話，間接摧毀他的人生，他選擇用 **職場霸凌 + 制度流程** 報復你，而不是直接溝通。

---

## 1. 登場角色（固定 / 人數）

### 核心登場（6 位）
1. **主角（你）**
2. **HR（神秘人X / 真兇）**
3. **同事A（執行型霸凌者 / 也可能被逼迫）**
4. **主管（制度背書者 / 也可能被逼迫）**
5. **同事C（關係操控者 / 也可能被逼迫）**
6. **同事B（同事群代表 / 輿論與孤立的放大器）**

### 支援登場（可選，1–2 類型即可）
- **旁人/證人**（茶水間、會議室、群聊裡的那個「看見但不說」的人）
- **客戶/合作方代表**（用來讓主管/HR「必須要一個替罪羊」）

> 設定：同事群「背景存在」，但**主要用同事B代表出場/對話/出手**，避免推理變散。

---

## 2. 霸凌鏈條：每個人「為何做壞事」（動機結構）

> 每個角色都要有「自己的理由」＋「HR 可控制的把柄/利誘」，玩家才能在玩法上拆解。

### HR（神秘人X / 真兇）
- **表面臉**：中立、程序、規範、保護公司
- **深層動機（單一事件 / 2016）**：
  - 他需要你一句**背書/作證**（推薦、證明清白、或替他說一句話就夠）
  - 你拒絕，並留下「他情緒不穩/不要找我背書」這類**標籤句**（你覺得是自保/合理）
  - 後果：他失去關鍵機會、被圈內默默黑名單，人生向下崩（你完全不記得這件事）
  - 他相信：你不是沒能力幫，是你選擇不幫 → 以為你「選擇性冷血」→ 長期怨恨
- **手段**：
  - 用制度切割責任（考核、申訴、紀錄、離職流程）
  - 用利益與恐懼驅動 A/C/主管（升遷、獎金、背鍋、把柄）
  - 用「公司風向」驅動同事B/同事群（誰站你那邊誰倒楣）
- **玩家可拆點**：留下流程證據、卡住關鍵節點（挑戰 11 是主要入口）

#### 行動限制（你說的：不能明目張膽）
- HR **不會親自霸凌你**、不會在公開場合「下指令害你」；他永遠站在「依法辦理/公司風險」那邊。
- 他要的是：**每一步看起來都合理**，即使你喊冤，也會被當成「情緒化」「不服管理」。

#### 三層掩護（讓他永遠有退路）
1. **表面層（合法話術）**：匿名檢舉 / 績效與試用 / 稽核 / 程序瑕疵 / 公司政策
2. **中介層（代理人）**：主管背書、同事A執行、同事C情感操控、同事B帶風向
3. **痕跡層（可推理但不可直接定罪）**：同樣的模板用語、關鍵時間點「剛好」出現、流程節點總是卡你

#### 可觀測痕跡（給線索系統用）
- HR 發出的信件/紀錄有固定模板措辭（像是同一個人寫的）
- 關鍵事件前後，總會有「匿名檢舉」或「內部關切」當理由啟動流程
- 你提出證據時，對方第一反應永遠是「流程」「紀錄」「可受理範圍」而不是事實

#### 偷偷摸摸的核心：導引其他人「仇視你」（不下命令，只做敘事導引）
> HR 最狠的地方不是權力，是他能讓大家「以為自己是自發討厭你」。

- **策略1：微暗示（Seed）**
  - 對 A/主管/B 用一句話種子：「你覺得他最近是不是有點…不穩？」
  - 不提供結論，只提供「疑點」讓對方自己補完。
- **策略2：選擇性資訊（Selective truth）**
  - 只把對你不利的片段拿出來（截圖、會議紀要、情緒爆點），把前因後果消掉。
  - 讓主管/同事B覺得自己是在「風險控管」。
- **策略3：社會證明（Social proof）**
  - 用「有人反映」「大家都覺得」「不是只有你這樣看」逼你被貼上問題員工標籤。
  - 同事B負責把這句話變成群聊風向。
- **策略4：道德包裝（Moral cover）**
  - 把霸凌包裝成「提醒你」「幫你適應」「公司是為你好」。
  - 對外永遠站在「制度公平」那一側。
- **策略5：讓代理人覺得有利可圖（Incentive）**
  - 對同事A：績效/升遷暗示；對同事C：保護/封口；對主管：風險移轉；對同事B：站對邊更安全。

#### 導引仇視的可觀測痕跡（更好埋線索）
- 同一套標籤詞反覆出現：**情緒化 / 不服管理 / 不合群 / 影響團隊 / 風險**
- 你每次一反擊，群聊/流程「剛好」就出現新材料（像有人在等）

### 同事A（執行型）
- **自己的壞念頭（非被迫）**：嫉妒你、怕被取代、想搶功立功
- **被 HR 推一把（可控）**：HR 暗示「你不配合，升遷/績效/轉正會出問題」
- **做壞事類型**：示好套弱點 → 搶功/背鍋/散播 → 製造你「能力差」形象
- **玩家可拆點**：讓他失去「合理化」與「退路」（逼他在恐懼與罪惡感上露餡）

### 主管（制度背書者）
- **自己的壞念頭（非被迫）**：KPI/效率至上、怕麻煩、把人當可替換資源
- **被 HR 控制（可控）**：HR 掌握他流程瑕疵/用人違規/過去黑歷史，讓他不得不配合
- **做壞事類型**：放任不公、刻意忽視、用「程序」壓你、需要替罪羊時把你推出去
- **玩家可拆點**：用證據逼他回到「風險控制」思維（他最怕曝光）

### 同事C（關係操控者）
- **自己的壞念頭（非被迫）**：享受便利（情感/資源/名聲），但拒絕承擔責任
- **被 HR 利誘/威脅（可控）**：HR 提供保護/升遷/封口；或掌握他私生活把柄
- **做壞事類型**：冷暴力、羞辱、切斷你的支援、把你的情緒弱點當武器
- **玩家可拆點**：切斷他「既要便利又要安全」的路，逼他在恐懼上失控（挑戰 13）

### 同事B（同事群代表 / 放大器）
- **角色定位**：群聊核心、訊息轉發站、風向兵（同事群「背景存在」，但由他具體出手）
- **自己的壞念頭（非被迫）**：從眾、自保、八卦、享受「掌握消息」的優越感
- **被 HR 間接控制（可控）**：HR 塑造「你是問題員工」風向，讓他覺得孤立你是安全選擇；必要時用績效/人事評語壓他站隊
- **做壞事類型**：帶風向、轉傳片段、話術包裝成「大家都這樣說」、讓你在團體裡失去立足點
- **玩家可拆點**：用「公開證據 + 程序正確」逆轉風向，逼他收手甚至反咬（挑戰 12）

## 3. 故事流程（前情提要 → 14 時刻 → 真相）

### A. 前情提要（死亡回顧）要完成的任務
- 讓玩家感受到：**這不是單點惡意，而是一條霸凌鏈條**
- 埋兩個核心伏筆：
  - **「表面嫌疑人其實在害怕」**（被威脅）
  - **「制度/流程在關鍵時刻會踩死你」**（暗指 HR）

### B. 10 個時刻的節奏（精簡版主線）
- **前半（1–3）**：建立信任 → 被蒐集弱點（A/C/B 在做事）
- **中段（4–6）**：搶功/背鍋/關係操控 → 玩家開始反擊
- **後段（7–9）**：制度擊殺（HR/主管）→ 關鍵恐懼線索連鎖（11/12/13）
- **收束（10）**：所有線索指向幕後黑手 → 進入道德困境

---

## 4. 真相揭露（4 階段，對齊線索）

1. **誤導**：你以為是 A / C / 主管
2. **懸疑**：你發現他們都「被威脅」→ 幕後黑手存在
3. **震撼**：幕後黑手是「HR」，而你其實在 2016 年成為他人生轉折
4. **深度**：你不是無辜聖人，但他也不是正義使者 → 由玩家選擇復仇/原諒/揭露
