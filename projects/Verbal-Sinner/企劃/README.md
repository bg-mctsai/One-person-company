## 核心概念

**因為人生過得很慘，於是回到5年前，重新開始，並獲得系統幫助。**

## 遊戲流程

1. **前情提要（2分鐘）**：展現玩家悲慘人生的關鍵片段，系統出現
2. **決定復活**：玩家選擇復活，必須觀看廣告
3. **時空轉換**：回到5年前
4. **100次挑戰**：針對悲慘的事故一一反駁，改變過去
5. **成功復活**：完成所有挑戰，獲得全新未來

## 與舊版本的差異

- **舊版本**：穿書重生（進入書中世界）
- **新版本**：時光倒流（回到5年前的現實世界）
- **舊版本**：多個副本世界（玄幻、都市、宮廷等）
- **新版本**：單一時間線，針對過去事件進行反駁

## 後續擴展

- **副本系統**：原本的穿書重生內容將作為副本系統，後續再處理
- **對話系統**：挑戰系統的對話機制後續完善
- **付費機制**：廣告跳過、加速挑戰等付費選項
