## 🎯 核心概念

**因為人生過得很慘，於是回到5年前，重新開始，並獲得系統幫助。**

**故事核心**：5年前（2019年），你認識了職場同事、朋友、情人。他們一開始對你很好，但從2019年7月開始，他們一步步陷害你，讓你走向深淵。現在你要回到過去，改變這一切。

## 🎮 遊戲流程

1. **前情提要（2分鐘）**：展現玩家悲慘人生的關鍵片段，系統出現
2. **決定復活**：玩家選擇復活，必須觀看廣告
3. **時空轉換**：回到5年前（2019年）
4. **20次挑戰**：針對被陷害的事件一一反駁，改變職場同事、朋友、情人對你的看法
5. **成功復活**：完成所有20個挑戰，獲得全新未來
